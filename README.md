# VexTurningPoint2019

UML's Vex Turning Point code


## Building
### Install dependencies
You need:
* git
* make
* gcc-arm-none-eabi
* python3
* python3-pip

```shell
pip install pros-cli
```

Clone the repo
```shell
git clone https://github.com/UMLRoboticsClub/VexTurningPoint2019.git
```

## Build the project
```shell
cd VexTurningPoint2019
make
```

## Uploading your code
```shell
pros flash
```

## Open serial connection with cortex
```shell
pros terminal
```
